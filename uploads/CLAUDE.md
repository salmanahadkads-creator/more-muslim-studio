# More Muslim — Project Instructions

## Social Post Rules

These rules apply whenever designing social posts for More Muslim. They override general brand defaults where social-specific behaviour differs.

### Typography
- No muted text — all type at full opacity (no `opacity` < 1 on any text element)
- No italic — upright Roman only on social
- Caps tracking is universal — every uppercase string uses `letter-spacing: 0.165em`, including URLs
- Minimum sizes: headline `96px`, eyebrow/CTA labels `48px`, body copy `32px`
- Body line-height is `1.2` — not 1.4 or 1.5
- Text alignment is centred — no left-aligned social posts
- Use `text-wrap: balance` on body paragraphs

### Layout
- Distribute elements evenly across full canvas height (`justify-content: space-around`) — do not cluster into a centre block
- No decorative separators (no rules, dashes, or dividers between text elements)
- No large uniform padding containers — let spacing do the work

### Brand marks
- No horizontal logo lockup on social posts — symbol only as sign-off
- Symbol sits bottom centre, minimum `140px` on 1080×1350, full opacity

### Copy
- Never use the show tagline as body copy — all copy must be specific to the post subject
- Be concrete: describe the actual thing (the job, the episode, the event)
