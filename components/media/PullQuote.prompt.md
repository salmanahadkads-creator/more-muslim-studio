The brand's signature device: a large italic serif quotation. Pass a plain string and it wraps in curly quotes automatically.

```jsx
<PullQuote size="xl" cite="Aunty Galiema · Bo-Kaap">
  I want to let them know that there's consequences of everything.
</PullQuote>
```

`size` md/lg/xl, `align` left/center, `marks` to toggle auto quote-marks.
